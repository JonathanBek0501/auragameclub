Computer Club Cashier (local)
-----------------------------
Run: open index.html in your browser (works offline).
Files:
- index.html
- styles.css
- app.js

Features:
- 5 rooms with timers (start/stop)
- Add products (snacks/drinks)
- Add product to active room sessions
- Auto-calc time price (10 000 so'm/hour) and items price
- End session archives the result and resets the room
- Export archive to CSV
- Uses localStorage for persistence

Notes:
- This is a simple local solution. For multi-device or cloud backup, migrate to a server (Firebase or similar).
